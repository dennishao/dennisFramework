#DennisFramework

##简介
dennisframework是一款微型的php mvc框架，其目的在于让大家更好的理解php mvc框架.
##目录说明
	-applications           //应用文件目录
		-config             //配置文件
		-controllers        //控制器目录
		-models             //模型目录
		-views              //视图目录
	-framework              //框架目录
		-core               //框架核心目录
		-database           //框架数据库目录
		-helpers            //框架第三方函数目录
		-libraries          //框架第三方类库目录
	-public                 //公共文件目录
		-css                //css样式目录
		-images             //图片文件目录
		-js                 //js函数目录
		-uploads            //上传目录
	-index.php              //入口文件
