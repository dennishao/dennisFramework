#DennisFramework<br/>
##简介<br/>
dennisframework是一款微型的php mvc框架，其目的在于让大家更好的理解php mvc框架.<br/>

##目录说明<br/>
-applications          应用文件目录<br/>
         --config             配置文件<br/>
    --controllers               控制器目录<br/>
    --models                    模型目录<br/>
    --views                      视图目录<br/>
-framework                   框架目录<br/>
    --core                      框架核心目录<br/>
    --database                 框架数据库目录<br/>
    --helpers                 框架第三方函数目录<br/>
    --libraries               框架第三方类库目录<br/>
-public                       公共文件目录<br/>
    --css                     css样式目录<br/>
    --images                 图片文件目录<br/>
    --js                        js函数目录<br/>
    --uploads                 上传目录<br/>
-index.php                     入口文件<br/>
